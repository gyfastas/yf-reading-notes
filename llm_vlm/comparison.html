  • 多模态占比提升：从 <5% (早期) → 10-20% (Seed-VL, Llama 4, K2.5)
  • 质量 > 数量：20T 高质量数据 (Seed-VL) vs 36T 全量数据 (Qwen3)
  • 专业化数据：STEM/代码专用数据成为差异化关键 (GLM-5 SWE, Intern-S1 科学)

  • 视觉原生：从"图文拼接"走向"原生多模态 tokenization"

Post-Training 趋势：
  • 环境交互 RL 崛起：MiniMax Forge (200K 环境), Step Scalable RL — 从静态数据走向真实环境反馈
  • Process Reward 普及：M2.5 和 2512.07783 都验证了步骤级奖励的优势
  • CISPO > PPO for MoE：序列级 clipping 比 token 级更稳定
  • FP32 LM Head 是 MoE RL 硬约束：S1-Pro 和 MiniMax CISPO 独立发现同一问题

  • Zero-Vision SFT：K2.5 证明强联合预训练下 post-training 可极大简化
  • Interleaved Thinking：MiniMax M2 系列的交错思考模式（<think> 标签）

10 — 关键洞察

架构趋势

  • MoE 绝对主导：12 个模型中 8 个为 MoE，仅 Claude 4 / Qwen3-VL 坚持 Dense
  • 两种路线：大 MoE (K2 1T/32B, GLM 744B/40B) vs 效率 MoE (Step 196B/11B, MM 230B/10B)
  • MLA 扩散：DeepSeek MLA 被 Kimi-K2 直接采用
  • SWA 兴起：Step 的 3:1 SWA 是长上下文 + 效率的新方案

训练范式

  • Token Scaling 放缓：第一梯队 30T+，第二梯队 15T — 增量收益递减，质量/方法 > 数量
  • Mid-training 成标配：Qwen3/K2/Step 都开源了 mid-train checkpoint
  • RL 成为核心差异化：M2→M2.5 同架构纯 RL 提升 15 ppt SWE-Bench，Step/K2.5 也重度投入 RL
  • 环境交互 RL 趋势：从 preference data → 真实环境 task completion reward

效率竞争

  • "Small Active, Big Brain"：10-11B active 足以达到 frontier 性能 (Step AIME 97.3, MM SWE 80.2)
  • MTP 推理加速：4-token prediction 成为标准 (Step MTP-3, DSV3 MTP)
  • 训练时量化：K2.5 QAT INT4 在训练中引入量化噪声，部署 2x 加速无损
  • 成本颠覆：开源 MoE ($0.15-0.30/1M input) vs 闭源 API ($3-15/1M) — 10-50x 差距

多模态融合

  • 原生联合预训练主流化：K2.5 / Llama 4 / Gemini 2.5 都采用原生多模态
  • K2.5 Zero-Vision SFT 启示：联合预训练足够强 → post-training 可极简化视觉数据
  • 视频处理收敛：4 帧压缩 (K2.5) / 动态 FPS (Qwen3-VL) — 时序压缩是关键
