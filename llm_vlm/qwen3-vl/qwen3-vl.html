Qwen3-VL Technical Report

Alibaba Cloud | 36T Tokens | 119 Languages | 256K Context

📌 TL;DR 核心规格

Pre-training Data: 36 trillion tokens
语言支持: 119种语言和方言
上下文: 256K tokens (原生)
OCR支持: 39种语言
关键创新：
  • 三阶段预训练 (General → Reasoning → Long Context)
  • RoPE base frequency 1,000,000 (ABF)
  • Dual Chunk Attention (DCA) + YARN
  • 动态FPS视频采样

🎯 三阶段 Pre-training

阶段 数据类型 Token数 序列长度 目标
S1: General 通用语料 + 多语言文本 ~30T 4,096 语言能力、世界知识
S2: Reasoning STEM、代码、合成数据 ~5T 4,096 增强推理能力
S3: Long Context 高质量长上下文语料 数百B 32,768 长上下文 (4x扩展)

🔄 Mid-training / 长上下文扩展

长上下文训练分布

长度范围 比例
