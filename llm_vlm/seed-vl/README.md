## 核心信息

### 模型规格
- **参数规模**: — (待补充)
- **架构**: Dense/MoE
- **上下文长度**: 128K
- **预训练数据**: ~20T tokens

### 预训练数据特色

| 维度 | 规格 |
|------|------|
| 总 Tokens | ~20T |
| 序列长度 | 128K |
| 图像数据 | 数万亿图像-文本对 |
| 视频数据 | 视频帧序列训练 |
| 多模态占比 | ~20% (图文+视频) |
| 代码数据 | ~25% |
| STEM 数据 | ~15% |

### 数据管线亮点
- **视觉感知清洗**: 基于视觉理解的文档解析
- **多尺度训练**: 支持 224px 到 4K 分辨率的渐进训练
- **联合去重**: 图像+文本跨模态去重
- **质量过滤**: CLIP 相似度 + 人工校验

### 关键技术
- 视觉为中心的预训练策略
- 原生多模态联合训练
- 视频理解能力（帧序列训练）
