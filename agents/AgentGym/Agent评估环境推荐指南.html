1. 概述

AgentGym 提供了一个包含 14个环境、89个任务类型 的综合评估平台。本指南基于论文中的实验设计，为不同研究目标提供环境选择和评估集构建的推荐。

6. 实用建议

6.1 环境选择检查清单

  • 研究目标明确：是测试通用能力还是特定能力？
  • 计算资源评估：长程环境（如SciWorld）需要更多计算资源
  • 评估时间预算：完整AgentEval（1160实例）需要约1小时测试时间
  • 基线对比需求：是否需要与AgentEvol等方法对比？
  • 数据需求：是否需要使用AgentTraj进行预训练？

6.2 常见误区避免

  1. 不要只选简单环境：MAZE和Wordle虽然简单，但无法代表Agent的通用能力
  2. 不要忽略工具环境：Weather、Movie等工具环境是测试Agent实用能力的重要组成
  3. 注意环境依赖性：部分环境需要外部API（Weather、Movie），确保网络连接
  4. 考虑评估成本：WebArena使用真实网站，可能需要更多调试时间
